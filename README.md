# A Taste of Well-Being — Isha Yoga Centre Recipes

A beautiful static website featuring **101 recipes** from the book *A Taste of Well-Being* by Isha Foundation.

## 🌐 Live Site
Host on GitHub Pages: `https://<your-username>.github.io/<repo-name>/`

## 📁 Files
- `index.html` — Main page
- `style.css` — All styles
- `recipes-data.js` — All 101 recipes + glossary data
- `app.js` — Search, filter and modal logic

## 🚀 Deploy to GitHub Pages
1. Create a new GitHub repository
2. Upload all files in this `recipes/` folder to the repo root
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at `https://<username>.github.io/<repo>/`

## ✨ Features
- Search recipes by name or ingredient
- Filter by category
- Morning / Evening / Meal time tags on every recipe
- Full ingredients, step-by-step method, tips
- Health benefits section for every recipe
- Ingredient glossary with Ayurvedic benefits
- Sadhguru's food wisdom section
- Kitchen techniques guide
- Fully responsive — works on mobile
